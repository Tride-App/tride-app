# Tride App

Basic Next.js web client for Tride.

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3000` by default.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
